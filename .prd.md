# PRD - 修复 release-check.sh awk 模式 bug

Priority: P1

## 背景

release-check.sh 中的 awk 模式有 bug：范围模式 `/### C1:/,/^###/` 的开始和结束条件在同一行匹配，导致只提取到标题行而非完整的证据块。

## 目标

修复 awk 模式，使其能正确提取从 `### C{N}:` 到下一个 `### ` 之间的完整内容。

## 验收标准

- release-check.sh 能正确识别 .layer2-evidence.md 中的 HTTP_STATUS
- C5-001 回归测试通过
- CI 全绿
