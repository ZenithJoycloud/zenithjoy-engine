# PRD - 清理 develop 上的 PRD/DoD 文件

## 问题

PR #265 错误地将功能分支文件（.prd.md, .dod.md, .quality-evidence.json, .quality-gate-passed）合并到了 develop。

这些文件应该只存在于功能分支（cp-*, feature/*），不应该在 develop/main 分支。

## 目标

从 develop 分支删除这些文件。

## 范围

**删除文件**：
- .prd.md
- .dod.md
- .quality-evidence.json
- .quality-gate-passed

**不影响**：
- 其他代码文件
- 已有的功能

## 验收标准

- [ ] develop 分支上不存在 .prd.md
- [ ] develop 分支上不存在 .dod.md
- [ ] develop 分支上不存在 .quality-evidence.json
- [ ] develop 分支上不存在 .quality-gate-passed
