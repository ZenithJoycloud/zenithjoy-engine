# PRD: Release 模式 QA-DECISION 修复

## 背景

Release 模式 L2A 检查要求 QA-DECISION.md 包含 `Decision: PASS`。
当前文件使用 `Decision: NO_RCI` 导致 release-check 失败。

## 修复方案

将 QA-DECISION.md 的 Decision 字段从 `NO_RCI` 改为 `PASS`。
这是 release 模式的强制要求，确保 CI 能正确通过。

## 验收标准

- [x] QA-DECISION.md 包含 `Decision: PASS`
- [x] release-check CI 通过
